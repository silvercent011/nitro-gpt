# Nitro GPT

## Descrição

Template de uma api rest utilizando [nitro](http://nitro.unjs.io) configurada com serviços da OpenAI

## Instalação

Escolha uma das opções abaixo para instalar as dependências e rodar a api:

### NPM

```bash
npm install
npm run dev
```

### Yarn

```bash
yarn install
yarn dev
```

### PNPM

```bash
pnpm install
pnpm dev
```

### Bun

```bash
bun install
bun dev
```
